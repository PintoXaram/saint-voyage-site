# Saint Voyage

**Monuments for the Senses** — Sculptural vessels and transportive fragrances. Where scent becomes ritual, atmosphere, and emotional architecture.

## Stack

- **Next.js 15.3** with React 19 and TypeScript
- **Tailwind CSS v4** for styling
- **GSAP** for scroll-driven animations
- **Playfair Display** + **DM Sans** typography

## Page Sections

1. **Navbar** — Site navigation
2. **HeroSection** — Hero with background video
3. **ScrollVideoSection** — Scroll-animated product video
4. **ScentSection** — Fragrance details
5. **PricingSection** — Product pricing with 3 color variants (beige, charcoal, terracotta)
6. **Footer**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command         | Description              |
|-----------------|--------------------------|
| `npm run dev`   | Start development server |
| `npm run build` | Production build         |
| `npm run start` | Start production server  |
| `npm run lint`  | Run ESLint               |
